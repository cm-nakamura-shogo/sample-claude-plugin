今ステージングされている差分について、commitメッセージを考えてcommit作成までを行います。

commitメッセージは日本語を使います。

メッセージ形式は、Conventional Commitに基づいたものを使用してください。

リモートへのpushは行わない様にしてください。（ユーザ自身にpushさせるように促します）

なおコミットメッセージにClaude Codeによるものであることがわかる署名は不要です。
